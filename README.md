# Forkify – Recipe Search App

## Description
Forkify is a modern, responsive web application that allows users to **search over 1,000,000 recipes**, view detailed recipe information, adjust servings, and bookmark favorite recipes.  
Built with **HTML, SCSS, and JavaScript**, it follows a **modular architecture** using the **MVC pattern**, state management, and responsive design.

---

## Features
- 🔍 **Search Recipes:** Search by recipe name or ingredient.  
- 📄 **Recipe Details:** View ingredients, cooking time, servings, and directions.  
- ⚖️ **Dynamic Servings:** Adjust servings and see ingredient quantities update automatically.  
- ⭐ **Bookmark Recipes:** Save favorite recipes.  
- 📱 **Responsive Design:** Fully mobile-first and responsive UI.  
- 🔢 **Pagination:** Navigate search results easily.  
- ➕ **Add Recipe Modal:** A modal to input new recipes (demo/static).  

---

## Tech Stack
- **Frontend:** HTML5, SCSS, JavaScript (ES6+)  
- **Bundler:** Parcel  
- **API:** Forkify API (`https://forkify-api.jonas.io`)  
- Live Demo(`forkify-eyad.netlify.app`)
